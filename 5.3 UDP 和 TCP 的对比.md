一、UDP
```bash
	1.UDP面向无连接
	2.UDP使用尽最大努力交付（不保证可靠）
	3.UDP是面向报文的
		对于应用层下发的报文，既不合并也不拆分，而是保留这些报文的边界
	4.UDP没有拥塞控制
	5.UDP支持单播、多播和组播（一对一，一对多，多对一，多对多）的交互通信
	6.UDP的首部开销小，只有 8 byte（B/字节)
```

二、TCP
```bash
	1.TCP面向连接的运输层协议
	2.每一条TCP连接只能有两个端点（点对点连接）
	3.TCP提供可靠交付的服务
	4.TCP提供全双工通信
	5.TCP面向字节流
	6.TCP的首部开销最小 20 B ，最大 60 B
```
